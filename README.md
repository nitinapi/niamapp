# Aanu & Ajja's Gift to Niam

This app is Aanu and Ajja's gift to Niam. It helps him pick between fun activities today and provides a chronicle of his life to look back on in the future.

Upcoming Features
- Help Niam choose a fun activity for today.
- Preserve memories so he can read a chronicle of his life later.

Instructions:

Run server in one terminal: 
run PS C:\Users\initi\niamApp\backend> mvn spring-boot:run

Start UI in another terminal:
run start c:\Users\initi\niamApp\index.html

Enjoy!
